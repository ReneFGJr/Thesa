As licenças Creative Commons são uma forma de proteger os direitos autorais de uma obra, permitindo que o autor escolha como deseja compartilhar sua criação. As licenças CC oferecem diferentes níveis de permissão, desde o uso total da obra até restrições específicas.

As principais licenças CC incluem:

BY Os créditos devem ser dados ao criador.
SA Permite adaptações da obra, desde que sejam compartilhadas pela mesma licença.
ND Permite redistribuição, comercial ou não, mas não permite modificações.
NC Permite uso não comercial, com atribuição, mas não permite uso comercial.
