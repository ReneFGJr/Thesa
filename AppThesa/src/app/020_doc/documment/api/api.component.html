Glossário do Thesa

Conceito

Um conceito é uma ideia, noção ou representação mental que expressa o significado de algo.
Ele é abstrato e independente da linguagem.
Na organização do conhecimento, o conceito é a base que dá sentido aos termos usados para descrevê-lo.
É por meio dos conceitos que se estruturam as relações entre informações em sistemas como tesauros, ontologias e classificações.

Termo

Um termo é uma palavra ou expressão que representa um conceito específico em um determinado contexto.
Os termos são usados para nomear e descrever conceitos, facilitando a comunicação e a compreensão entre pessoas que compartilham o mesmo domínio de conhecimento.
Os termos podem variar de acordo com o idioma, a cultura e o contexto em que são utilizados.
Em sistemas de organização do conhecimento, como tesauros e ontologias, os termos são associados a conceitos para garantir que a informação seja acessível e compreensível.

Uso das API

API abertas
Endpoint Método Descrição
/api/thopen GET Lista todos os tesauros publicos.
/api/th/:th GET Dados detalhados de um tesauro.
  • :th - Número (ID) do tesauros (Thesa)
/api/terms/:th GET Conceitos do tesauro.
  • :th - Número (ID) do tesauros (Thesa)
  • :apikey (opcional) - Se informado retorna variável editMode (true ou false)
/api/t/:idT GET Termo dentro dos tesauros.
  • :idT - Número (ID) do termo
/api/c/:idC GET Conceitos do tesauro.
  • :idC - Número (ID) do conceito
API de manutenção do tesauro
/api/createThesa POST
  • :apikey - API key do usuário
  • :title - Título do instrumento de controle de vocabulário
  • :acronic - Abreviatura, acronico do instrumento
  • :type - Tipo do instrumento
    • 1: Vocabulário controlado / Glossário
    • 2: Tesauro (tradicional - TG/TE/TR)
    • 3: Tesauro (SKOS/semântico)
  • :visibility - Conteúdo a ser salvo dentro do campo de descrição
    • 1: Restrito
    • 2: Público
  • :finality - Conteúdo a ser salvo dentro do campo de descrição
    • 1: Testar a ferramenta
    • 2: Atividade de ensino/pesquisa
    • 3: Produção de tesauro
/api/removeRelation POST
  • :apikey - API key do usuário
  • :terms - ID dos termos a serem relacionados
  • :thesaID - Número (ID) do tesauro
  • :verb - Verbo de relacionamento
    • altLabel: Termo alternativo
    • hiddenLabel: Termo oculto
/api/relateTerms POST
  • :apikey - API key do usuário
  • :terms - ID dos termos a serem relacionados
  • :thesaID - Número (ID) do tesauro
  • :verb - Verbo de relacionamento
    • altLabel: Termo alternativo
    • hiddenLabel: Termo oculto
/api/saveDescription POST
  • :apikey - API key do usuário
  • :field - Nome do campo da descrição (title, achronic, introduction, Methodology, Audience, License)
  • :thesaurus - Número (ID) do tesauro
  • :description - Conteúdo a ser salvo dentro do campo de descrição
/api/relateConcept POST
  • :apikey - API key do usuário
  • :thesaurus - Número (ID) do tesauro
  • :property - Tipo de vinculo (broader, related)
  • :c1 - Conceito principal
  • :c2 - Conceito secundario
/api/term_add POST
  • :apikey - API key do usuário
/api/concept_create_term POST
  • :apikey - API key do usuário
/api/term_list/:idTh POST
  • :apikey - API key do usuário
  • :idTh - ID do tesauro
/api/term_pref_list/:idTh/:idC POST
  • :apikey - API key do usuário
  • :idTh - ID do tesauro
  • :idC - ID do conceito
/api/saveNote POST
  • :apikey - API key do usuário
  • :field - Nome do campo da descrição (title, achronic, introduction, Methodology, Audience, License)
  • :thesaurus - Número (ID) do tesauro
  • :noteID (opcional) - ID da nota para alteração
  • :conceptID - Número (ID) do conceito
  • :note - Conteúdo a ser salvo dentro do campo de
  • :noteType - Tipo de vinculo (broader, related)
  • :thesaurus - Número (ID) do tesauro
  • :lang - Idioma da nota (por, eng, spn, fra, get, ...)
/api/deleteNote POST
  • :apikey - API key do usuário
  • noteID: - ID do registro da Nota
/api/tools POST
  • :apikey - API key do usuário
/api/import POST
  • :apikey - API key do usuário
