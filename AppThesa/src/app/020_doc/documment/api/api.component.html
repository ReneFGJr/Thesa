Introdução

Bem-vindo à API do nosso sistema. Esta documentação apresenta os principais recursos.

Autenticação

Use um token JWT para autenticar suas requisições.

Endpoints

  • GET /users
  • POST /login
  • DELETE /users/:id

Exemplos de Uso

curl -X GET https://api.meusistema.com/users \
  -H "Authorization: Bearer seu_token_aqui"

Tratamento de Erros

Erros seguem o formato padrão HTTP com mensagens descritivas.

Uso das API

API abertas
Endpoint Método Descrição
/api/thopen GET Lista todos os tesauros publicos.
/api/th/:th GET Dados detalhados de um tesauro.
  • :th - Número (ID) do tesauros (Thesa)
/api/terms/:th GET Conceitos do tesauro.
  • :th - Número (ID) do tesauros (Thesa)
/api/t/:idT GET Termo dentro dos tesauros.
  • :idT - Número (ID) do termo
/api/c/:idC GET Conceitos do tesauro.
  • :idC - Número (ID) do conceito
API de manutenção do tesauro
/api/term_add POST
  • :apikey - API key do usuário
/api/concept_create_term POST
  • :apikey - API key do usuário
/api/term_list POST
  • :apikey - API key do usuário
/api/tools POST
  • :apikey - API key do usuário
/api/import POST
  • :apikey - API key do usuário
API estatíticas
/api/status GET Saude do sistema.
/api/resume GET Dados resumidos de todos os tesauros.
